## 📌 Overview

A classic 2-player Tic-Tac-Toe game built for the command line. This project was developed to practice control flow, input validation, array manipulation, and game state management in Python.

## ✨ Features

- **Dynamic Board:** The grid updates in real-time after every turn.
- **Input Validation:** Prevents game crashes from invalid inputs (like letters or out-of-bound numbers).
- **Automated Win/Draw Detection:** Efficiently checks 8 winning combinations and draw states without bloated `if/else` chains.
- **Clean Console UI:** Automatically clears the terminal screen between turns for a smooth gameplay experience.

## 🚀 How to Play

1. Navigate to this directory in your terminal.
2. Run `python main.py`.
3. Players take turns entering a number (1-9) to place their 'X' or 'O' on the corresponding grid spot.
4. The first to align 3 marks horizontally, vertically, or diagonally wins!
