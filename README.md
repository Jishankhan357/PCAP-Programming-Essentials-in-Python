## 📌 About This Repository

Welcome to my Python mini-project portfolio! This repository serves as a centralized archive of the console-based applications I developed while mastering the fundamentals of Python through the **PCAP: Programming Essentials in Python** certification course.

The core objective of this repository is to demonstrate the practical application of theoretical concepts, including:

- Control Flow & Logic Building
- Data Structures (Lists, Dictionaries)
- File Handling & Binary Serialization (`pickle`)
- Modular Programming & Architecture
- Command Line Interface (CLI) Design
- REST API Integration & JSON Parsing
